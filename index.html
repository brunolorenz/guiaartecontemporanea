Token do GitHub

O token permite que o guia salve seu progresso diretamente no repositório. Ele fica armazenado só no seu navegador — nunca vai ao servidor.

1. Acesse github.com/settings/tokens/new
2. Note: "Guia Arte Contemporânea"
3. Expiration: No expiration
4. Marque apenas: repo (acesso completo ao repositório)
5. Clique "Generate token" e copie aqui
